# family-bills (React + TS + Chakra + Supabase)

## 1) Instalar
```bash
npm i
```

## 2) Configurar .env
Crie `.env` na raiz:
```env
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```

## 3) Rodar
```bash
npm run dev
```

## 4) SQL (Supabase)
Cole e rode no SQL Editor o script que acompanha na conversa.
